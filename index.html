Quick Setup

Get started in under 5 minutes

1

Install env-sync

Install using our automated script or Go directly:

curl -fsSL https://raw.githubusercontent.com/lliamscholtz/env-sync/main/scripts/install.sh | bash

Or with Go:

go install github.com/lliamscholtz/env-sync@latest
2

Install Dependencies

Automatically install Azure CLI and verify your setup:

env-sync install-deps
env-sync doctor
3

Generate Encryption Key

Create a secure encryption key for your team:

env-sync generate-key

Share this key securely with your team through your password manager.

4

Initialize Project

Set up your project configuration:

export ENVSYNC_ENCRYPTION_KEY="your-base64-key" env-sync init --vault-url https://your-vault.vault.azure.net/ --secret-name myapp-env --key-source env
5

Start Syncing

Push your initial .env file and start file watching:

env-sync push

Your .env file is now securely synced with Azure Key Vault!

env-sync watch
